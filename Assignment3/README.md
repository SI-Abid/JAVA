## Calculator 


### Description 

    Simple calculator with basic operations.

### Prototype

```
---------------------------------------
| ----------------------------------- |
| | 1 + 2 + 5                       | |
| |                              8  | |
| ----------------------------------- |
---------------------------------------
| -------------------------  -------  |
| |       |       |       |  |  +  |  |
| |   7   |   8   |   9   |  |-----|  |
| |-------|-------|-------|  |  -  |  |
| |       |       |       |  |-----|  |
| |   4   |   5   |   6   |  |  x  |  |
| |-------|-------|-------|  |-----|  |
| |       |       |       |  |  /  |  |
| |   1   |   2   |   3   |  |-----|  |
| |-------|-------|-------|  |  %  |  |
| |       |       |       |  |-----|  |
| |   0   |   .   |  DEL  |  |  =  |  |
| -------------------------  -------  |
---------------------------------------
```
