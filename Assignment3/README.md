# Calculator  

## Description  

    Simple calculator with basic operations.

### Prototype

```python
---------------------------------------
| ----------------------------------- |
| | 1 + 2 + 5                       | |
| |                              8  | |
| ----------------------------------- |
---------------------------------------
| -------------------------  -------  |
| |       |       |       |  |  +  |  |
| |   7   |   8   |   9   |  |-----|  |
| |-------|-------|-------|  |  -  |  |
| |       |       |       |  |-----|  |
| |   4   |   5   |   6   |  |  *  |  |
| |-------|-------|-------|  |-----|  |
| |       |       |       |  |  /  |  |
| |   1   |   2   |   3   |  |-----|  |
| |-------|-------|-------|  |  %  |  |
| |       |       |       |  |-----|  |
| |   0   |   .   |  DEL  |  |  =  |  |
| -------------------------  -------  |
---------------------------------------
```
